1. Capture Target Window (F8)

Run the program and press the F8 key to capture the current active window.

2. Choose Configuration Option

After capturing the window, choose one of the following options:

  • Enter '1' to load a file.
  • Enter '2' to manually add keys and delays.

3. Load Configurations

If you enter 1, write the file name and hit enter, the program will load configurations from a file. To continue, follow the instructions displayed in the console.

4. Manually Add Keys and Delays

If you enter 2 you can manually add keys and their corresponding delays. Enter the key you want to simulate and the delay in milliseconds, seconds ('s'), or minutes ('m') when prompted. enter "0" To stop adding keys, enter 'exit'.

5. Toggle Keypress Simulation

Type 'start' to begin simulating keypresses. Type 'stop' to pause the keypress simulation.

6. Add or Remove Keys During Simulation

To add a specific number of new keys during simulation, type 'add' followed by the number (e.g., 'add 5'). To remove a specific number of keypress threads, type 'remove' followed by the number (e.g., 'remove 3').

7. Save or Exit the Program

To save your key configurations, type 'save' and enter the filename. To exit the program, type 'exit'. This will stop the keypress simulation and close the application.

Notes:

  • The program accepts delay input in milliseconds by default.
  • To specify delays in seconds, add 's' after the delay (e.g., '5s' for 5 seconds).
  • To specify delays in minutes, add 'm' after the delay (e.g., '2m' for 2 minutes).
  • The program runs in the command prompt and may require administrator privileges for certain operations.
  • Ensure that you have selected the correct target window for key simulation.

Example Usage:

  • Press F8 to capture the window.
  • Choose 1 to load configurations or 2 to manually add keys (enter '0') to stop adding keys.
  • If loading configurations, follow console instructions.
  • If adding keys manually, follow the prompts to add keys and set delays.
  • Type 'start' to begin keypress simulation.
  • Type 'add 5' to add key press for the number 5 during simulation.
  • Type 'remove 3' to remove key press for the number 3 during simulation.
  • Type 'stop' to pause the simulation.
  • Type 'save' to save configurations or 'exit' to exit the program.
