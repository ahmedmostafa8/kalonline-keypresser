1. Capture Target Window

Run the program and wait for the instruction: "Press F8 to capture the target window..." Press the F8 key to capture the current active window.

2. Add Keys and Delays

After capturing the window, you can add keys and their corresponding delays. Enter the key you want to simulate and the delay in milliseconds, seconds ('s'), or minutes ('m') when prompted. To stop adding keys, enter '0'.

3. Toggle Keypress Simulation

Type 'start' to begin simulating keypresses. Type 'stop' to pause the keypress simulation.

4. Add or Remove Keys During Simulation

To add a specific number of new keys during simulation, type 'add' followed by the number (e.g., 'add 5'). To remove a specific number of keypress threads, type 'remove' followed by the number (e.g., 'remove 3').

5. Exit the Program

To exit the program, type 'exit'. This will stop the keypress simulation and close the application.

Notes:

  • The program accepts delay input in milliseconds by default.
  • To specify delays in seconds, add 's' after the delay (e.g., '5000s' for 5 seconds).
  • To specify delays in minutes, add 'm' after the delay (e.g., '2m' for 2 minutes).
  • The program runs in the command prompt and may require administrator privileges for certain operations.
  • Ensure that you have selected the correct target window for key simulation.
  • Be cautious when automating keypresses, as it may violate terms of service for some applications.

Example Usage:

  • Capture window by pressing F8.
  • Add keys and delays (e.g., 'A' with a delay of 1000 milliseconds or 'B' with a delay of '5s' for 5 seconds).
  • Type 'start' to begin keypress simulation.
  • Type 'add 5' to add five more keypress threads during simulation.
  • Type 'remove 3' to remove three keypress threads during simulation.
  • Type 'stop' to pause the simulation.
  • Type 'exit' to exit the program.
